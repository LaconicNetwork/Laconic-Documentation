# Laconic-Documentation

This repository is the source for the site https://docs.laconic.com, detailing the usage of the [Laconic Network](https://laconic.com).

## Local editing

The site is built with [Hugo](https://github.com/gohugoio/hugo/). To edit locally, first install Hugo, and then start Hugo in server mode in the root of your checkout of this repository.
